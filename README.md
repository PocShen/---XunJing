# XunJing

寻径 v1.0，一款面向实战的 Burp Suite URL 重定向探测插件。

它强调被动探测、低门槛使用和真实场景命中能力，适合在正常浏览、登陆、退出、注册、回调、OAuth、SSO 等流程中自动识别潜在的 URL 重定向问题。

## 核心特性

- 被动流量探测
- Query / Body / JSON / Header / Path 多位置检测
- 缺失参数自动补参
- 内置高风险参数名字典
- OAuth / SSO / callback / logout / register 场景优化
- 多跳跳转链分析
- Burp Dashboard / Issues 联动
- 支持 JDK8 与 JDK11+ 双版本

## 适用场景

- SRC 挖洞
- 认证流程测试
- 登陆登出功能测试
- 回调与中转地址检查
- Burp 被动审计
